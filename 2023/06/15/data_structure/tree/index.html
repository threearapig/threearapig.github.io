树

关于树：

n 个结点的有限集合（n >= 0）

n = 0 时，此树称为 空树

n > 0 时，有且仅有一个 根结点（Root）

  • n > 1 时，除了根结点还有其余结点
  • 其余结点可分为 m 个互不相交的有限集
  • 每个有限集又是一棵树
  • 这样的有限集就称为根的 子树

结点的分类

结点拥有的子树的个数，称为该结点的度

度为 0 的结点称为 叶结点 或 终端结点

度不为 0 的结点称为 非终端结点 或 分支结点

对于度不为 0 的结点，除 根结点 之外，分支结点也可以称为 内部结点

树的度是树内各结点的度的最大值

什么是分支？
如果将树画出来，两个结点之间的连结，或者那条连线，可以称之为 分支

那么分支结点呢？
就是这个结点，它往下还有分支，也就是还有连线（不管是多少连线）

这里往下还有的分支（也就是连线），是不包含此结点与其双亲结点的连线的

结点间的关系

一个结点的 子树的根 称为该结点的 孩子
相应的，该结点称为该孩子的 双亲

对于很多结点，如果它们的双亲为同一个，那么这些结点就称为 兄弟结点

一个结点的 祖先 是从 根结点开始 到该结点 所经分支上的所有结点

对于一个树，它的所有结点（不包括根结点），都是 根结点 的 子孙

其他相关概念

结点的 层次，意思就是该结点在这个树结构中，处于第几层
对于层数的划分：

  • 根结点为第一层
  • 根的孩子为第二层
  • 根的孩子的孩子为第三层
  • 依次类推, 直到叶节点为止

一棵树中，如果一些结点的 双亲 结点处于 同一层次，那么就称这些结点为 堂兄弟

树的深度/高度：树中结点的最大层次

什么是森林？

森林是 m 棵 互不相交 的树的集合（m >= 0）

一棵树如果将其 根结点 去除，那么剩下来的 子树，就可以构成一个森林

同样的，如果将很多森林的 根结点，使这些森林的根结点都拥有 同一个双亲，那么就可以构成一棵树

一棵树可以是一个森林
但是
一个森林不能构成一棵树

因为森林没有一个统一的 根结点

二叉树

对于二叉树，它拥有树的很多特性，但是有几点和树不同

每个结点最多有两个子树

注意这里是最多有两个子树

左子树和右子树的顺序不能颠倒

根结点左侧的子树为左子树，右侧的子树为右子树

即使树中某结点只有一颗子树，也要区分它是左子树还是右子树

对于拥有子树的结点，它的子树是有左右之分的，也就是子树明确分为： 左子树 和 右子树

但是对于普通的树
当一个结点拥有两个子树时，对为了将这两个子树区分开来，就将它们称呼为左子树和右子树
但是这只是一个为了区分它们的称呼，实际上并没有硬性要求这样称呼（也可以将它们分为 A/B 子树来区分）

当一个结点只拥有一个子树时，此时不需要区分子树了，因为就一个子树，因此也就没有左右子树一说

特殊二叉树

为什么需要知道这两种特殊二叉树？
因为它们在顺序存储方式下可以复原

满二叉树

满二叉树：
树的深度为 k，且有 2k - 1 个结点的二叉树，称为二叉树
在一颗二叉树中，如果所有分支结点都存在左子树和右子树，并且所有叶子节点都在同一层上

特点：

  1. 叶子只能出现在最下面一层
  2. 非叶子结点的度一定是 2
  3. 在同样深度的二叉树中，满二叉树的结点个数最多，叶子树最多

完全二叉树

完全二叉树：
对一颗具有 n 个结点的二叉树按层序编号，编号为 i 的结点与同样深度的满二叉树中编号为 i 的结点在二叉树中的位置完全相同

这里注意是编号相同，不是结点值相同

二叉树的性质

性质1

在二叉树的第 i 层最多有 2i - 1 个结点（i >= 1）

性质2

深度为 k 的二叉树至多有 2k - 1 个结点（k >= 1）

性质3

对任何一颗二叉树，如果其终端结点数为 n0，度为 2 的结点数为 n2，则 n0 = n2 + 1

性质4

具有 n 个结点的完全二叉树的深度为 $\lfloor \log_2 10 \rfloor$ + 1（$\lfloor x \rfloor$ 表示不大于 x 的最大整数，也就是向下取整）

性质5

如果对一棵有 n 个结点的完全二叉树（其深度为 $\lfloor \log_2 10 \rfloor$ + 1）的结点按层序编号（从第 1 层到第 $\lfloor \log_2 10 \rfloor$ + 1 层，每层从左到右），对任一结点 i（1 $\le$ i $\le$ n）有：

  1. 如果 i = 1, 则结点 i 是二叉树的根，无双亲; 如果 i > 1，则其双亲是结点 $\lfloor i/2 \rfloor$

  2. 如果 2i > n，则结点 i 无左孩子（结点/为叶子结点）; 否则其左孩子是结点 2i

  3. 如果 2i + 1 > n，则结点/无右孩子; 否则其有孩子是结点 2i + 1
